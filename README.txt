directions for reading your card

download bday2022.zip (you have already done this lol) and unzip
    you may need to download The Unarchiver from the app store to unzip
save benbday2022 folder to Desktop (it is probably in Downloads)
open Terminal
    by default you will open in your home directory
    type "cd Desktop"
    type "ls" ... make sure you see benbday2022 listed
type "cd benbday2022"
type in "sudo chmod 755 benbday2022.sh"
enter your computer password
type in "./benbday2022.sh -h"
